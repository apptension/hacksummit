put /user/{id}/evaluation

update the evaluation

  • Request
  • Response

URI Parameters

  • id: required (integer)

Body

Type: application/json

Form Parameters
  • evaluationId: (integer)

    Id of previously received evaluation.

  • state: (integer)

    Rating status. 0 = pending, 1 = answered, 2 = skipped (dismiss). Optional if rating is not dismissed.

  • starred: (integer)

    Rating value. 0 = zero, 1 = plus one. Optional if rating is dismissed.

  • comment: (string)

    Optional comment.

HTTP status code 200

Body

Type: application/json

Example:

{
  "success": true
}
